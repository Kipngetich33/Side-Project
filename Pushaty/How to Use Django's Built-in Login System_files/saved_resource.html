                                                                                                                                                                Vitor Freitas Mod Petr Svetr • 2 years ago

                                                                                                                                                                Hi Petr,
                                                                                                                                                                You can pretty much place the base.html template anywhere. You just have to refer to the right location when you are extending the base template. If you place it inside an app folder, you perhaps can do it like this:

                                                                                                                                                                {% extends 'app_name/base.html' %}

                                                                                                                                                                {% block title %}Login{% endblock %}

                                                                                                                                                                {% block content %}
                                                                                                                                                                    <h2>Login</h2>
                                                                                                                                                                      <form method="post">
                                                                                                                                                                            {% csrf_token %}
                                                                                                                                                                            {{ form.as_p }}
                                                                                                                                                                            <button type="submit">Login</button>
                                                                                                                                                                      </form>
                                                                                                                                                                {% endblock %}

                                                                                                                                                                And in the settings, you have to make sure you are telling Django to search for templates inside app folders:

                                                                                                                                                                TEMPLATES = [
                                                                                                                                                                    {
                                                                                                                                                                    'BACKEND': 'django.template.backends.django.DjangoTemplates',
                                                                                                                                                                            'DIRS': [os.path.join(BASE_DIR, 'mysite/templates')],
                                                                                                                                                                            'APP_DIRS': True,
                                                                                                                                                                            'OPTIONS': {
                                                                                                                                                                                    'context_processors': [
                                                                                                                                                                                            'django.template.context_processors.debug',
                                                                                                                                                                                            'django.template.context_processors.request',
                                                                                                                                                                                            'django.contrib.auth.context_processors.auth',
                                                                                                                                                                                            'django.contrib.messages.context_processors.messages',
                                                                                                                                                                                        ],
                                                                                                                                                                                },
                                                                                                                                                                        },
                                                                                                                                                                ]

                                                                                                                                                                        Vitor Freitas Mod Petr Svetr • 2 years ago

                                                                                                                                                                        Ah! Now I understood your question. Indeed it is correct.

                                                                                                                                                                        To achieve that, inside the login app, you would need to create a new folder:

                                                                                                                                                                        login/templates/login/home.html
                                                                                                                                                                        login/templates/login/login.html
                                                                                                                                                                        login/templates/login/logged_out.html

                                                                                                                                                                        And then use it like this:

                                                                                                                                                                        url(r'^login/$', auth_views.login, {'template_name': 'login/login.html'}, name='login'),
                                                                                                                                                                        url(r'^logout/$', auth_views.logout, {'template_name': 'login/logged_out.html'}, name='logout'),

                                                                                                                                                                        While it is correct and recommended, it also brings some challanges in the beginning, for those who are learning Django

                                                                                                                                                                        for the first time. So to avoid the overhead I simplified the example.

                                                                                                                                                                        Now usually my take on that matter:

                                                                                                                                                                        The templates/ folder in the project root, I place all the templates that are shared across all my apps,

                                                                                                                                                                        such as base.html, pagination.html, form.html and so on. Also, I add folders for registration and

                                                                                                                                                                        admin or any other template override from third party library.

                                                                                                                                                                        In the app_name/templates/app_name/ folder, I place the app specific templates. So I can use common

                                                                                                                                                                        names such as index.html or new.html without problems.

                                                                                                                                                                                      # Add here the path to the template dirs
                                                                                                                                                                                  ),
                                                                                                                                                                              ...

                                                                                                                                                                          Basically you should have 'APP_DIRS': True and the correct full path to the 'DIRS' template config.
